How It Works

This page uses JavaScript to detect your mobile operating system by analyzing your browser's user agent string.

The detection script identifies:

  • Android devices
  • iOS devices (iPhone, iPad, iPod)
  • Windows Phone devices
  • Other/unknown devices

Instead of automatically redirecting, we display a relevant link based on your device type. This approach gives you control over your navigation.

// Detection function example:
function getMobileOperatingSystem() {
  var userAgent = navigator.userAgent;
  if (/windows phone/i.test(userAgent)) {
    return "Windows Phone";
  }
  if (/android/i.test(userAgent)) {
    return "Android";
  }
  if (/iPad|iPhone|iPod/.test(userAgent)) {
    return "iOS";
  }
  return "unknown";
}
